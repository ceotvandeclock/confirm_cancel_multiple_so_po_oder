This Module helps you to Confirm/Cancel Multiple Sale and Purchase order in Odoo Apps.




Confirm Multiple Sale Orders.




Bellow you can see select "Confirm" State and Click on Confirm Button.




Bellow, you can see this selected Sale orders is Confirmed.




Cancel Multiple Purchase Order.




Bellow you can see select "Cancel" State and Click on Confirm Button.




Bellow, you can see this selected Purchase Orders is Canceled.
